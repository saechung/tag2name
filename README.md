# 🎵 tag2name

> **MP3 메타데이터(ID3 태그) 기반 파일명 자동 변경 도구**

지저분하거나 규칙 없는 MP3 파일 이름을 음악 파일 내부에 저장된 *노래 제목(ID3 태그)**을 추출하여 `곡제목.mp3`로 깔끔하게 정리해 주는 자동화 스크립트

---

## ✨ 주요 기능
- **태그 기반 변경:** 파일명이 아무리 망가져 있어도 내부 태그의 노래 제목을 읽어 정확한 파일명으로 변경합니다.
- **안전한 복사 방식:** 원본 파일(`old`)을 훼손하거나 지우지 않고, 이름을 바꾼 복사본을 `new` 폴더에 생성합니다.
- **원클릭 자동화:** 가상환경(`venv`) 생성부터 필수 패키지 설치, 스크립트 실행까지 스크립트 하나로 해결합니다.
- **폴더 자동 생성:** 작업에 필요한 폴더가 없으면 프로그램이 알아서 생성하고 안내합니다.

---

## 📂 폴더 구조
스크립트를 실행하면 아래 구조로 폴더가 자동 구성됩니다.
```
tag2name/
├── auto_rename.py   # 메인 파이썬 소스 코드
├── start.bat        # 윈도우용 실행 스크립트
├── start.sh         # 리눅스/맥용 실행 스크립트
├── .gitignore       # 깃허브 업로드 제외 설정 파일
├── LICENSE          # MIT 라이선스
├── old/             # ⚠️ 변환할 원본 MP3 파일을 넣는 곳 (자동 생성)
└── new/             # 🎉 변환 완료된 파일이 저장되는 곳 (자동 생성)
```
---

## 🚀 사용 방법

### 💻 Windows 환경
이 저장소를 다운로드하거나 복제(git clone)합니다.

start.bat 파일을 더블 클릭하여 실행합니다. (최초 실행 시 old, new 폴더가 자동 생성됩니다.)

새로 생성된 old 폴더 안에 이름 변경을 원하는 MP3 파일들을 집어넣습니다.

다시 start.bat을 더블 클릭하면 모든 파일이 변환되어 new 폴더에 저장됩니다.

### 🐧 Linux / macOS 환경
터미널을 열고 프로젝트 폴더로 이동한 뒤 아래 명령어를 입력합니다.

#### 1. 스크립트 실행 권한 부여 (최초 1회만)
chmod +x start.sh

#### 2. 스크립트 실행 (가상환경 세팅 및 변환 진행)
./start.sh
